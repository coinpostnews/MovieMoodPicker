# Movie Mood Picker

Веб-приложение, которое поможет вам выбрать фильм в соответствии с вашим текущим настроением. Вы выбираете настроение с помощью кнопок с эмодзи, а наш ИИ предложит 3-5 подходящих фильмов с деталями, такими как постер, описание, рейтинг и возможные платформы для просмотра.

## Основная ценность

Быстрый и простой способ найти фильм, который соответствует вашему текущему настроению — без долгого поиска и мучительного выбора.

## Возможности

-   **Рекомендации фильмов по настроению:** Получайте персонализированные предложения фильмов, просто выбрав эмодзи, отражающее ваше настроение.
-   **Предложения на базе ИИ:** Использование ИИ для анализа вашего настроения и предоставления релевантных вариантов фильмов.
-   **Полная информация о фильмах:** Каждая рекомендация включает основную информацию, такую как постер фильма, краткое описание, его рейтинг и потенциальные платформы для просмотра.
-   **Простота и скорость:** Приложение разработано для быстрого взаимодействия и немедленных результатов.

## Технологический стек

Этот проект построен на современном стеке веб-разработки, который обеспечивает удобство для разработчиков и масштабируемость, при этом учитывая первоначальные ограничения проекта по быстрой разработке.

-   **Фронтенд:** [React 19](https://react.dev/) — декларативная, компонентная библиотека пользовательского интерфейса.
-   **Сборщик:** [Vite 8](https://vitejs.dev/) — молниеносный инструмент сборки и сервер разработки.
-   **Стилизация:** [Tailwind CSS 4.3](https://tailwindcss.com/) — утилитарный CSS-фреймворк для быстрой разработки пользовательского интерфейса.
-   **Безопасность типов:** [TypeScript 5.x](https://www.typescriptlang.org/) — для повышения качества кода и удобства поддержки.
-   **Интеграция с ИИ:** [Groq API (бесплатный тариф)](https://console.groq.com/docs) — для быстрого вывода LLM для анализа настроения.
-   **Данные о фильмах:** [TMDb API v3 (бесплатный)](https://developer.themoviedb.org/docs) — для получения полной информации о фильмах.
-   **Загрузка данных:** [@tanstack/react-query 5](https://tanstack.com/query/latest) — для эффективной загрузки, кеширования и управления состоянием данных.

### Примечание к архитектурному решению

Хотя первоначальная спецификация проекта предполагала использование одного HTML-файла с встроенными CSS и JS для максимально быстрой разработки, мы выбрали настройку на базе React + Vite. Это решение обеспечивает лучшую основу для поддержки, масштабируемости и удобства разработки в долгосрочной перспективе, что соответствует современным практикам веб-разработки.

## Развертывание

Приложение разработано для легкого развертывания на статических хостингах, таких как GitHub Pages.

## Начало работы

### Необходимые условия

-   Node.js (рекомендуется v18.x или v20.x)
-   npm или Yarn

### Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/movie-mood-picker.git
    cd movie-mood-picker
    ```
2.  Установите зависимости:
    ```bash
    npm install # или yarn install
    ```
3.  Настройте переменные окружения:
    Создайте файл `.env` в корне проекта и добавьте ваши ключи API:
    ```
    VITE_GROQ_API_KEY=ВАШ_КЛЮЧ_GROQ_API
    VITE_TMDB_API_KEY=ВАШ_КЛЮЧ_TMDB_API
    ```
    *(Примечание: для этого MVP с бесплатными API-ключами они будут видны в клиентском коде при развертывании. Для продакшн-приложений рассмотрите возможность использования бэкенд-прокси для защиты ключей API.)*

### Запуск локально

```bash
npm run dev # или yarn dev
