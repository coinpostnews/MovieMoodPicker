# Movie Mood Picker

## What This Is

Веб-приложение, которое подбирает фильмы по настроению пользователя. Пользователь выбирает настроение через кнопки с эмодзи, AI анализирует запрос и предлагает 3-5 подходящих фильмов с деталями (постер, описание, рейтинг, где посмотреть).

## Core Value

Быстрый и простой способ найти фильм, который соответствует текущему настроению — без долгого поиска и выбора.

## Context

**Технический стек:**
- Frontend: Vanilla JS (один HTML файл со всем кодом)
- AI: Groq API (бесплатный, быстрый LLM для анализа настроения)
- Данные о фильмах: TMDb API (The Movie Database, бесплатный)
- Деплой: GitHub Pages (статический хостинг)

**Архитектура:**
- Простой single-page application без фреймворков
- Клиентский код делает запросы напрямую к API (Groq + TMDb)
- Никакого бэкенда — всё работает в браузере
- API ключи будут в коде (для MVP приемлемо, т.к. бесплатные сервисы)

**Пользовательский флоу:**
1. Открывает сайт → видит кнопки с настроениями
2. Кликает на настроение (например, "Весело 😄")
3. Видит лоадер "Подбираю фильмы..."
4. Видит список из 3-5 фильмов с постерами
5. Кликает на фильм → видит детальную карточку
6. Может вернуться назад и выбрать другое настроение

## Constraints

- **Время**: Быстрая разработка — проект должен быть готов за несколько часов
- **Простота**: Никаких сложных архитектур — один HTML файл с inline CSS и JS
- **Бесплатность**: Только бесплатные API и хостинг
- **Деплой**: Должен легко деплоиться на GitHub Pages без настройки CI/CD
