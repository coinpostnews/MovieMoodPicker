## Валидация AI результатов

### Question 1: Как проверять существование фильмов в TMDb?

| Option | Description | Selected |
|--------|-------------|----------|
| Fuzzy match (Рекомендую) | Для каждого названия от Groq делать TMDb search, брать первый результат если совпадение > 80% | ✓ |

**User's choice:** Fuzzy match (Рекомендую)

### Question 2: Что делать если после валидации осталось < 3 фильмов?

| Option | Description | Selected |
|--------|-------------|----------|
| Retry с Groq (Рекомендую) | Запросить у Groq ещё фильмы (max 2 retry), затем показать что есть | ✓ |
| Показать что есть | Показать что прошло валидацию, даже если < 3 фильмов | |
| Показать ошибку | Показать ошибку пользователю и предложить выбрать другое настроение | |

**User's choice:** Retry с Groq (Рекомендую)

### Question 3: На каком языке искать фильмы в TMDb API?

| Option | Description | Selected |
|--------|-------------|----------|
| Оба языка (Рекомендую) | Искать на русском и английском параллельно, брать лучший результат | |
